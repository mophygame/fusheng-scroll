# 男主人設素材

每條路線放入一張今生與一張前世透明 WebP：

```text
zhaoxue-modern.webp
zhaoxue-ancient.webp
fenxin-modern.webp
fenxin-ancient.webp
tingyue-modern.webp
tingyue-ancient.webp
wuxiang-modern.webp
wuxiang-ancient.webp
shanhe-modern.webp
shanhe-ancient.webp
```

建議規格：

- 畫布：`1200 × 1800 px`
- 透明背景 WebP
- 頭頂保留約 5%
- 人物裁至大腿中段
- 今生與前世使用相同畫布及人物位置
